# TOEIC 900+ 衝刺

專為衝刺 TOEIC 900 分以上設計的網頁學習軟體。針對「目前 600–750、考前 1–2 個月、每天 1 小時」的備考情境，採**三階段週期化訓練**，涵蓋單字、聽力（Part 2/3/4）、文法（Part 5/6）、閱讀（Part 7）。

## 如何開啟

直接雙擊 `index.html` 即可使用（聽力需要 Edge 或 Chrome，使用系統內建語音合成，免音檔、免網路）。

或啟動本地伺服器：

```
python -m http.server 8642
```

然後開啟 http://localhost:8642

## 三階段學習計畫（依考試日期自動切換）

在首頁設定考試日期後，每日菜單自動依階段生成：

| 階段 | 時期 | 每日 60 分鐘菜單 |
|---|---|---|
| 🧱 基礎期 | 考前 5 週以前 | 單字 15 分＋Part 5 十題＋聽力 Part 2 十題＋Part 6 一篇＋Part 7 一篇 |
| ⚙️ 強化期 | 考前 2–5 週 | 單字＋Part 5 十題＋聽力 Part 3/4 兩組＋Part 7 兩篇＋錯題重考 5 題 |
| 🚀 衝刺期 | 考前 2 週 | 單字（新字減半）＋Part 5＋聽力混合＋Part 7 限時＋錯題本清零 |

## 功能

- **單字卡**：181 個 900+ 進階詞彙，SM-2 間隔重複（SRS），自動發音
- **聽力 Part 2**：24 題應答問題，語音合成朗讀（不顯示文字），專攻間接回應與同字陷阱，答後看逐字稿
- **聽力 Part 3/4**：8 組對話/短講，男女聲自動區分，先讀題再聽，答完附逐字稿
- **Part 5/6**：70 題文法＋5 篇段落填空，選項洗牌、中文詳解、類別弱點統計
- **Part 7**：7 組單篇/雙篇/三篇閱讀，倒數計時器訓練答題節奏
- **錯題本**：全科目答錯自動收錄（聽力題重考時重新播放音檔），答對才移除
- **統計**：預估分數（L+R，樣本足夠才顯示）、14 天熱圖、聽力/文法類別正確率、單字進度
- **題庫/備份**：一鍵產生 AI 擴充指令 → 貼回 JSON 即匯入合併（自動驗證格式、去除重複）；學習紀錄＋自訂題庫可匯出成備份檔、一鍵還原，超過 7 天未備份會在首頁提醒

## 資料儲存與備份

- 每次作答**即時自動存檔**到瀏覽器 localStorage（學習紀錄 `toeic900`、自訂題庫 `toeic900_bank`）
- 「題庫/備份」分頁可匯出 JSON 備份檔（含學習紀錄＋自訂題庫）、匯入還原；超過 7 天未備份，首頁會出現提醒
- 換瀏覽器或清除瀏覽資料前，請先匯出備份

## 擴增題庫

「題庫/備份」分頁 →「一鍵擴增題庫」：選類型與數量 → 複製自動產生的 AI 指令 → 貼給 Claude（或任何 AI）→ 把回覆的 JSON 貼回來按匯入。系統會自動驗證格式、剔除與現有題目重複的內容，匯入後立即加入練習輪替。

也可以直接編輯 `js/data/` 下的檔案擴充內建題庫。注意：詳解文字請引用選項內容而非 (A)(B)(C) 字母，因為選項顯示時會隨機洗牌。

## 檔案結構

```
js/data/part6.js      Part 6 題庫（5 篇）
js/data/part7.js      Part 7 題庫（7 組）
```
