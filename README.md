# TOEIC 900+ 衝刺

專為衝刺 TOEIC 900 分以上設計的網頁學習軟體。針對「目前 600–750、考前 1–2 個月、每天 1 小時」的備考情境，採**三階段週期化訓練**，涵蓋單字、聽力（Part 2/3/4）、文法（Part 5/6）、閱讀（Part 7）。

## 🌐 線上使用（電腦、手機都能開）

**https://realsnova.github.io/toeic-900/**

直接用手機或電腦的瀏覽器開啟這個網址即可，不用安裝任何東西。建議在手機瀏覽器把它「加入主畫面」，開啟起來就像 App 一樣。

> 預設狀態下，每台裝置的學習紀錄各自獨立存放（存在該瀏覽器的 localStorage）。想讓手機與電腦顯示同一套進度，請設定下方的「雲端同步」。

### ✅ 真機確認清單

自動化測試涵蓋不到的幾件事，建議實際用手機/電腦跑一次：

- [ ] 手機瀏覽器聽力腔調：「我的→設定」看看裝置實際列出哪些腔調（iOS/Android 通常內建多腔調，效果如何請實際聽聽看）
- [ ] 手機「加入主畫面」：確認圖示是靶心圖案（不是網頁截圖），加入後全螢幕開啟像真的 App
- [ ] 鍵盤 Tab 導覽：真人手指離開觸控板、純用 Tab／Shift+Tab 走一遍，確認焦點框看得清楚（自動化工具在這台環境測不出真實焦點樣式，需要人眼確認）
- [ ] 淺色／深色主題在你實際螢幕上的對比與可讀性
- [ ] 完整模擬考跑一次全程（含真實語音速度，不是加速測試版本），感受聽力節的緊湊節奏

## 本機開啟（開發用）

直接雙擊 `index.html` 即可使用（聽力需要 Edge 或 Chrome，使用系統內建語音合成，免音檔、免網路）。

或啟動本地伺服器：

```
python -m http.server 8642
```

然後開啟 http://localhost:8642

## 三階段學習計畫（依考試日期自動切換）

在首頁設定考試日期後，每日菜單自動依階段生成：

| 階段 | 時期 | 每日 60 分鐘菜單 |
|---|---|---|
| 🧱 基礎期 | 考前 5 週以前 | 單字 15 分＋Part 5 十題＋聽力 Part 2 十題＋Part 6 一篇＋Part 7 一篇 |
| ⚙️ 強化期 | 考前 2–5 週 | 單字＋Part 5 十題＋聽力 Part 3/4 兩組＋Part 7 兩篇＋錯題重考 5 題 |
| 🚀 衝刺期 | 考前 2 週 | 單字（新字減半）＋Part 5＋聽力混合＋Part 7 限時＋錯題本清零 |

## 功能

- **介面**：首頁／練習／模擬考／錯題本／我的 5 大分區，手機底部導覽列、電腦頂部導覽列；淺色／深色雙主題（可跟隨系統或手動切換）
- **題庫規模**：全題庫逾 1,000 題（單字 621、Part 5 200、Part 6 40 格、Part 7 44 題、聽力 Part 2 60、Part 3/4 39 題），可再透過「一鍵擴增」持續增加
- **單字卡**：621 個 900+ 進階詞彙，SM-2 間隔重複（SRS），自動發音
- **聽力 Part 2**：60 題應答問題，語音合成朗讀（不顯示文字），專攻間接回應與同字陷阱，答後看逐字稿
- **聽力 Part 3/4**：13 組對話/短講，男女聲自動區分，先讀題再聽，答完附逐字稿
- **聽力腔調**：可在「我的→設定」選擇裝置上偵測到的腔調（美/英/澳等），或開啟「混合腔調」模式讓每題／每組隨機切換腔調，仿真正式考試混腔的情況；只列出裝置實際可用的腔調，若裝置只有一種會自動隱藏選單
- **Part 5/6**：200 題文法＋10 篇段落填空，選項洗牌、中文詳解、類別弱點統計
- **Part 7**：12 組單篇/雙篇/三篇閱讀，倒數計時器訓練答題節奏
- **模擬考**：仿真 TOEIC 兩節測驗——聽力節（Part 2 ×10＋Part 3/4 ×3 組，共 19 題，音檔自動連播、不可暫停或回放）＋閱讀節（Part 5 ×15＋Part 6 ×1 篇＋Part 7 ×2 組，限時 22 分鐘，可標記🚩待回頭、跳題、題號導航）。可選完整模考或單節模式。作答中不顯示對錯，交卷後產出成績報告（預估 L/R 分數、各 Part 正確率、弱項 Top 3 直接連到對應練習），錯題自動進錯題本，歷次成績以折線圖追蹤趨勢。組卷會優先抽「未出過的題」，題庫輪完一圈才重新洗牌，確保多次模考儘量不重複
- **錯題本**：全科目答錯自動收錄（聽力題重考時重新播放音檔），答對才移除
- **統計**：預估分數（L+R，樣本足夠才顯示）、14 天熱圖、聽力/文法類別正確率、單字進度
- **題庫/備份**：一鍵產生 AI 擴充指令 → 貼回 JSON 即匯入合併（自動驗證格式、去除重複）；學習紀錄＋自訂題庫可匯出成備份檔、一鍵還原，超過 7 天未備份會在首頁提醒
- **新手引導**：全新使用者（本機與雲端都沒有任何紀錄）首次開啟會出現 3 步設定（目標分數／考試日期／每日可投入時間），完成後自動排定每日新字量；已有紀錄的使用者不會看到，可隨時點背景略過
- **穩定性**：任何未預期錯誤都會用提示訊息呈現、不會讓頁面整個當掉；離線時會提示「先存本機，恢復網路後自動同步」；「我的」頁面底部有版本號與更新紀錄

## 資料儲存與備份

- 每次作答**即時自動存檔**到瀏覽器 localStorage（學習紀錄 `toeic900`、自訂題庫 `toeic900_bank`）
- 「題庫/備份」分頁可匯出 JSON 備份檔（含學習紀錄＋自訂題庫）、匯入還原；超過 7 天未備份，首頁會出現提醒
- 換瀏覽器或清除瀏覽資料前，請先匯出備份

## ☁️ 雲端同步設定（讓手機與電腦進度一致）

同步用 [Firebase](https://firebase.google.com)（Google 的免費雲端資料庫）。只需要一個 Google 帳號，不用信用卡，設定約 3 分鐘：

1. 前往 [Firebase 主控台](https://console.firebase.google.com/)，用 Google 帳號登入，按「新增專案」，輸入任意名稱（如 `toeic-sync`），一路下一步建立完成
2. 進入專案後，左側選單「建構」→「Realtime Database」→「建立資料庫」→ 選一個離你近的地區 → **安全性規則選「測試模式」開始**（之後會替換成下面更嚴謹的規則）
3. 資料庫建立後，切到「規則」分頁，貼上以下內容並發布：
   ```json
   {
     "rules": {
       "sync": {
         "$code": {
           ".read": true,
           ".write": true
         }
       }
     }
   }
   ```
   （這代表「知道同步代碼的人才能讀寫該代碼底下的資料」——足夠應付個人讀書進度，不建議存放機敏資料）
4. 回到專案總覽（點左上角齒輪 →「專案設定」），往下捲到「你的應用程式」，按 `</>`（網頁）圖示新增一個網頁應用程式，名稱隨意，**不需要**勾選 Firebase Hosting
5. 建立後會顯示一段 `firebaseConfig` 物件，**把這 7 個值整個複製給我（貼在對話裡）**，我會幫你寫進 `js/firebase-config.js` 並重新部署

設定完成後，在任一裝置的「題庫/備份」分頁按「產生新代碼」，把代碼抄到第二台裝置貼上並按「套用」，兩台裝置就會自動同步（存檔後約 1.5 秒推送雲端，開啟頁面時自動抓取最新進度）。

> Firebase 網頁設定值（apiKey 等）本來就設計成可公開，安全性是由上面的資料庫規則把關，不是靠隱藏這些值，所以放心提供給我寫進程式碼。

## Windows 安裝英式/澳式語音包

TOEIC 聽力會混用美/英/澳/加拿大等腔調，但電腦能提供哪些腔調完全看作業系統裝了什麼語音。手機（iOS/Android）通常開箱就有多種英文腔調可用；**Windows 電腦預設常常只有系統顯示語言對應的那一種**（例如只有中文語音、0 個英文語音），需要手動加裝語言包才會出現在「我的→設定→聽力腔調」選單中：

1. 開啟「設定」→「時間與語言」→「語言與地區」
2. 「偏好的語言」旁按「新增語言」，搜尋並加入「English (United Kingdom)」（英式）或「English (Australia)」（澳式）等
3. 安裝過程中若出現選項，**務必勾選「文字轉語音」（Text-to-speech）功能元件**，否則只會裝鍵盤/顯示語言，不會有語音
4. 安裝完成後重新整理本網頁，「我的→設定」的腔調選單就會出現新裝的腔調

## 擴增題庫

「題庫/備份」分頁 →「一鍵擴增題庫」：選類型與數量 → 複製自動產生的 AI 指令 → 貼給 Claude（或任何 AI）→ 把回覆的 JSON 貼回來按匯入。系統會自動驗證格式、剔除與現有題目重複的內容，匯入後立即加入練習輪替。

也可以直接編輯 `js/data/` 下的檔案擴充內建題庫。注意：詳解文字請引用選項內容而非 (A)(B)(C) 字母，因為選項顯示時會隨機洗牌。

```
js/data/listening.js     聽力題庫（Part 2 ×60、Part 3/4 ×13 組）
js/data/part5.js         Part 5 題庫（200 題）
js/data/part6.js         Part 6 題庫（10 篇）
js/data/part7.js         Part 7 題庫（12 組）
```
